myFlix Documentation

This is a server side API for myFlix. myFlix is a movie data base where users will be able to find information of different movies. Such information includes directors, genres, rating and much more. The application uses REST API, Javascript, Node.js and Express. The REST API will use standard HTTP syntax such as GET, POST, PUT and DELETE.

Request URL HTTP Method Request body data format Response
Get a list of all movies /movies GET None A JSON object holding data about all the movies
Get data about a single movie /movies/:title GET None A JSON object holding data about a specific movie { "Title": "Star Wars: The Empire Strikes Back", "Genre": { "Name": "Sci-Fi", } "Director": { "Name": "George Lucas", } }
Add a user /users POST A JSON object holding data to add a user, structured like: { "Name": "Matt Browning", "favoriteMovies": [ "Lion King", "Hot Fuzz" ] } A JSON object holding data about that the user was added, including an ID { "id": "919", "Name": "Matt Browning", "favoriteMovies": [ "Lion King", "Hot Fuzz" ] }
Remove a user by ID /users/:id DELETE None A text message confirming that the user was succesfully deleted
Allow users to update their favorite movies /movies/:id/:movieTitle POST None A text message confirming the movies have been added
Allow users to remove movies from their favorites /movies/:id/movieTitle DELETE None A text message confirming the movies were removed
